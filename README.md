# 🎮 Квест-комната | Аналитика выручки + ИИ прогноз

Веб-приложение для отслеживания выручки квест-комнаты с графиками, статистикой по дням недели и ИИ-прогнозом с учетом погоды и праздников.

## ✨ Возможности

- 📊 **Аналитика выручки** - общая, средняя, за 30 дней
- 📈 **Графики динамики** - 7/30/90/180 дней
- 📅 **Статистика по дням** - средние значения по дням недели
- 🔮 **Умный прогноз** - на ближайшие недели с учетом:
  - 🎉 Праздников
  - ☁️ Погоды в Екатеринбурге
  - 📊 Исторических данных
- 🤖 **ИИ-помощник** - отвечает на вопросы о выручке

## 🚀 Демо

https://quest-analytics.onrender.com

## 🛠️ Технологии

- Backend: Flask, Pandas, Matplotlib, Scikit-learn
- Frontend: HTML, CSS, JavaScript
- API погоды: Open-Meteo (бесплатно)
- ИИ: ApiFreeLLM
- Хостинг: Render
